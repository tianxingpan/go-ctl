# go-ctl
go-ctl是一款SSH的工具客户端辅助工具，支持同时操作多台远程机器执行命令，或者批量上传文件等等，是一款不不错的辅助工具。

## 子命令
### cli
控制远程机器执行命令，并返还结果。
#### 参数说明
##### command
远程执行的命令，支持短命名*-c*和长命名*--command*
```shell script
./go-ctl cli -c='ls /'
```
